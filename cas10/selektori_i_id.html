CSS Sintaksa

CSS dokument se sastoji od:

  • Selektora
  • Deklaracija
    1. Svojstava
    2. Vrednosti
h1 { color: blue; font-size: 15pt; }

CSS Komentari

Primer komentara u CSS-u je:
/* Ovo je komentar u CSS-u */
Komentari se otvaraju sa '/*', a zatvaraju se sa '*/'

Klase i Identifikatori

Klase (class) se koriste za grupisanje razlicitih elemenata koji imaju slicna svojstva.
Identifikatori (id) funkcionisu slicno kao i klase, ali se koriste samo za pojedinacne elemente.
